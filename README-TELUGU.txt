App Name: Sri Nidhi Study Hall
Package Name: com.srinidhi.studyhall
Website: https://srpkagitha-max.github.io/Sri-Nidhi-StudyHall/
Version: 1.0.0
Target SDK: 36

GITHUBలో BUILD చేయడం:
1. కొత్త private/public repository create చేయండి.
2. ఈ ZIP extract చేసి అన్ని files upload చేయండి.
3. Repository > Actions > Build Android App > Run workflow.
4. Build complete అయ్యాక Artifactsలో SriNidhi-APK-AAB download చేయండి.
5. app-debug.apk testing కోసం మాత్రమే.
6. app-release.aab ప్రస్తుతం unsigned release bundle; Play Store uploadకి signing అవసరం.

PLAY STOREకి ముందు:
- Google Play App Signing setup చేయాలి.
- Release keystoreతో signed AAB generate చేయాలి.
- Privacy Policyని public URLలో host చేయాలి.
- Store listing screenshots/feature graphic upload చేయాలి.

గమనిక: మొదటి Play Store releaseకు package name తర్వాత మార్చకూడదు.
